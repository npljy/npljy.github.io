з”ҹе‘Ҫе‘ЁжңҹиҜҰи§Ј

1.constructor(props, context)

constructorз”ҹе‘Ҫе‘ЁжңҹпјҢеҰӮдёҚйңҖиҰҒпјҢеҸҜзјәзңҒгҖӮйҖҡеёёдјҡеңЁ constructor ж–№жі•дёӯеҲқе§ӢеҢ– state е’Ңз»‘е®ҡдәӢд»¶еӨ„зҗҶзЁӢеәҸгҖӮ

дҪҶжҳҜпјҢеҰӮжһңеҶҷдәҶconstructorпјҢйӮЈд№Ҳеҝ…йЎ»еңЁе…¶дёӯи°ғз”Ёsuper(props);еҗҰеҲҷеҸҜиғҪдјҡеј•иө·жҠҘй”ҷгҖӮ

еҰӮ:

class Base extends Component {
    constructor(props) {
        super();  //еә”иҜҘдёә super(props);
    }
    state = {
        name: this.props.name
    }
    //....code
}

жҠӣеҮәејӮеёё: Uncaught TypeError: Cannot read property вҖҳnameвҖҷ of undefined.

еҗҢж ·пјҢеҰӮжһңе®ҡд№үдәҶcontext,еңЁstateдёӯйңҖиҰҒдҪҝз”Ёthis.contextеҺ»иҺ·еҸ–contextдёҠзҡ„еҶ…е®№пјҢеҲҷйңҖиҰҒsuper(props, context);

дёҚиҝҮпјҢеҰӮжһңдҪ зјәзңҒconstructor,йӮЈд№ҲеңЁstateдёӯпјҢеҸҜд»Ҙж”ҫеҝғзҡ„дҪҝз”Ё this.props жҲ–иҖ…жҳҜ this.contextпјҢдёҚдјҡеј•иө·жҠҘй”ҷгҖӮ

class Base extends Component {
    state = {
        name: this.props.name,
        color: this.context.color
    }
    //....code
}

еҲқе§ӢеҢ–зҡ„stateеҗҢж ·еҸҜд»ҘеңЁconstructorдёӯе®ҡд№үгҖӮеҰӮжһңйңҖиҰҒз»ҷж–№жі•з»‘е®ҡthisпјҢйӮЈд№Ҳз»ҹдёҖеңЁconstructorдёӯиҝӣиЎҢгҖӮ

2.static getDerivedStateFromProps(props, state)

еҪ“з»„д»¶зҡ„stateйңҖиҰҒж №жҚ®propsжқҘж”№еҸҳзҡ„ж—¶еҖҷеҸҜи°ғз”ЁжӯӨж–№жі•гҖӮиҝҷдёӘж–№жі•жҳҜеңЁ render() еүҚдјҡиў«жү§иЎҢпјҢжҜҸж¬Ўи§ҰеҸ‘renderеүҚпјҢйғҪдјҡи§ҰеҸ‘жӯӨж–№жі•гҖӮ

иҜҘж–№жі•жңүдёӨдёӘеҸӮж•°propsе’Ңstate; иҝ”еӣһеҖјдёәstateеҜ№иұЎ, дёҚйңҖиҰҒиҝ”еӣһж•ҙдҪ“stateпјҢжҠҠйңҖиҰҒж”№еҸҳзҡ„stateиҝ”еӣһеҚіеҸҜгҖӮеҰӮжһңдёҚйңҖиҰҒпјҢеҸҜд»Ҙиҝ”еӣһnull.

classBaseextendsComponent {
    state = {
        age:20
    }
    staticgetDerivedStateFromProps(props, state) {
        return {
            age:50
        }
    }
    render() {
        // 50return (
            <div>{this.state.age}</div>
        )
    }
}

иҝҷдёӘж–№жі•е…Ғи®ёз»„д»¶еҹәдәҺ props зҡ„еҸҳжӣҙжқҘжӣҙж–°е…¶еҶ…йғЁзҠ¶жҖҒ,д»Ҙиҝҷз§Қж–№ејҸиҺ·еҫ—зҡ„з»„д»¶зҠ¶жҖҒиў«з§°дёәжҙҫз”ҹзҠ¶жҖҒгҖӮеә”иҜҘи°Ёж…ҺдҪҝз”Ёжҙҫз”ҹзҠ¶жҖҒпјҢеҸҜиғҪдјҡеј•е…ҘжҪңеңЁзҡ„й”ҷиҜҜ

3.render

Reactз»„д»¶дёӯеҝ…йЎ»иҰҒжҸҗдҫӣзҡ„ж–№жі•гҖӮеҪ“stateжҲ–иҖ…propsд»»дёҖж•°жҚ®жңүжӣҙж–°ж—¶йғҪдјҡжү§иЎҢгҖӮ

render() жҳҜдёҖдёӘзәҜеҮҪж•°пјҢеӣ жӯӨпјҢдёҚиҰҒеңЁе…¶дёӯжү§иЎҢsetStateиҜёеҰӮжӯӨзұ»зҡ„ж“ҚдҪңгҖӮrenderеҝ…йЎ»жңүдёҖдёӘиҝ”еӣһеҖјпјҢиҝ”еӣһзҡ„ж•°жҚ®зұ»еһӢеҸҜд»Ҙжңү:

  • nullгҖҒStringгҖҒNumberгҖҒArrayгҖҒBooleanгҖӮ
  • React elements
  • Fragment
  • Portal

жіЁж„ҸдёҚиҰҒеңЁrenderдёӯи°ғз”ЁsetState

4.componentDidMount

componentDidMount()ж–№жі•жҳҜеңЁз»„д»¶еҠ иҪҪе®ҢеҗҺз«ӢеҚіжү§иЎҢпјҢд№ҹе°ұжҳҜеҪ“иҜҘз»„д»¶зӣёе…ізҡ„domиҠӮзӮ№жҸ’е…ҘеҲ°domж ‘дёӯж—¶гҖӮиҜҘж–№жі•еңЁз»„д»¶з”ҹе‘ҪдёӯеҸӘжү§иЎҢдёҖж¬ЎгҖӮ

дёҖиҲ¬жғ…еҶөпјҢжҲ‘д»¬дјҡеңЁиҝҷйҮҢsetState(),жҲ–иҖ…иҝӣиЎҢжҺҘеҸЈиҜ·жұӮпјҢи®ҫзҪ®и®ўйҳ…зӯүгҖӮ

classBaseextendsComponent {
    state = {
        age:20
    }
    componentDidMount() {
        this.fetchDate();
    }
    render() {
        return (
            <div>{this.state.age}</div>
        )
    }
    //other code
}

5.shouldComponentUpdate(nextProps, nextState, nextContext)

еңЁжёІжҹ“ж–°зҡ„propsжҲ–stateеүҚпјҢshouldComponentUpdateиў«и°ғз”ЁпјҢй»ҳи®Өиҝ”еӣһtrueгҖӮforceUpdate()ж—¶дёҚдјҡи°ғз”ЁжӯӨж–№жі•гҖӮ

еҰӮжһңshouldComponentUpdate()иҝ”еӣһfalseпјҢйӮЈд№ҲgetSnapshotBeforeUpdate,renderе’ҢcomponentDidUpdateдёҚдјҡиў«и°ғз”ЁгҖӮ

жӯӨз”ҹе‘Ҫе‘Ёжңҹдё»иҰҒз”ЁдәҺдјҳеҢ–жҖ§иғҪгҖӮ

6.getSnapshotBeforeUpdate(prevProps, prevState)

еңЁrender()зҡ„иҫ“еҮәиў«жёІжҹ“еҲ°DOMд№ӢеүҚиў«и°ғз”ЁгҖӮдҪҝз»„д»¶иғҪеӨҹеңЁе®ғд»¬иў«жӣҙж”№д№ӢеүҚжҚ•иҺ·еҪ“еүҚеҖјпјҲеҰӮж»ҡеҠЁдҪҚзҪ®пјүгҖӮиҝҷдёӘз”ҹе‘Ҫе‘Ёжңҹиҝ”еӣһзҡ„д»»дҪ•еҖјйғҪе°ҶдҪңдёәз¬¬дёүдёӘеҸӮж•°дј йҖ’з»ҷcomponentDidUpdate().

7.componentDidUpdate(prevProps, prevState, snapshot)

еңЁжӣҙж–°еҸ‘з”ҹеҗҺи°ғз”Ё componentDidUpdate()гҖӮеҪ“з»„д»¶жӣҙж–°ж—¶пјҢе°ҶжӯӨдҪңдёәдёҖдёӘжңәдјҡжқҘж“ҚдҪңDOMгҖӮеҰӮе°ҶеҪ“еүҚзҡ„propsдёҺд»ҘеүҚзҡ„propsиҝӣиЎҢжҜ”иҫғпјҲдҫӢеҰӮпјҢеҰӮжһңpropsжІЎжңүж”№еҸҳпјҢеҲҷеҸҜиғҪдёҚйңҖиҰҒзҪ‘з»ңиҜ·жұӮгҖӮ

еҰӮжһңз»„д»¶дҪҝз”Ё getSnapshotBeforeUpdate()пјҢеҲҷе®ғиҝ”еӣһзҡ„еҖје°ҶдҪңдёәз¬¬дёүдёӘвҖңеҝ«з…§вҖқеҸӮж•°дј йҖ’з»ҷ componentDidUpdate()гҖӮеҗҰеҲҷпјҢиҝҷдёӘеҸӮж•°жҳҜundefinedгҖӮ

8.componentWillUnmount()

еңЁз»„д»¶иў«еҚёиҪҪе№¶й”ҖжҜҒд№ӢеүҚз«ӢеҚіиў«и°ғз”ЁгҖӮеңЁжӯӨж–№жі•дёӯжү§иЎҢд»»дҪ•еҝ…иҰҒзҡ„жё…зҗҶпјҢдҫӢеҰӮдҪҝе®ҡж—¶еҷЁж— ж•ҲпјҢеҸ–ж¶ҲзҪ‘з»ңиҜ·жұӮжҲ–жё…зҗҶеңЁcomponentDidMountпјҲпјүдёӯеҲӣе»әзҡ„д»»дҪ•зӣ‘еҗ¬гҖӮ

жңҖеҗҺпјҢиҜҙжҳҺдёҖзӮ№:

componentWillMount,componentWillReceiveProps,componentWillUpdateиҝҷдёүдёӘз”ҹе‘Ҫе‘ЁжңҹеңЁReactжңӘжқҘзүҲжң¬дёӯдјҡиў«еәҹејғгҖӮ

иҖҢUNSAFE_componentWillUpdate,UNSAFE_componentWillReceiveProps,UNSAFE_componentWillUpdate жңӘжқҘзүҲжң¬дёӯд»ҚеҸҜз»§з»ӯдҪҝз”ЁгҖӮ

еҲқе§ӢеҢ–йҳ¶ж®ө(зҲ¶з»„д»¶е’Ңеӯҗз»„д»¶):

иҝҗиЎҢйҳ¶ж®ө:зҲ¶з»„д»¶props/stateжӣҙж–°

еӯҗз»„д»¶зҡ„shouldComponentUpdateиҝ”еӣһfalseпјҢеҲҷеӯҗз»„д»¶е…¶еҗҺзҡ„з”ҹе‘Ҫе‘ЁжңҹйғҪдёҚеңЁиҝӣиЎҢпјҢдҪҶжҳҜзҲ¶з»„д»¶зҡ„з”ҹе‘Ҫе‘Ёжңҹз»§з»ӯжү§иЎҢгҖӮ
