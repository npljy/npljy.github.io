React 渲染性能优化的三个方向，其实也适用于其他软件开发领域，这三个方向分别是:

  • 减少计算的量。 -> 对应到 React 中就是减少渲染的节点 或者 降低组件渲染的复杂度
  • 利用缓存。-> 对应到 React 中就是如何避免重新渲染，利用函数式编程的 memo 方式来避免组件重新渲染
  • 精确重新计算的范围。 对应到 React 中就是**绑定组件和状态关系, 精确判断更新的’时机’和’范围’.
